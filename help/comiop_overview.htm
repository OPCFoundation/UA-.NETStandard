COM Interoperability Overview

A benefit of OPC Unified Architecture is its ability to interoperate/integrate with OPC Classic technologies, such as DA 2.05a and DA 3.0 etc. A "gateway" service is provided with this package that will handle the integration for you.

The UA Configuration tool provides all of the options necessary to configure this gateway and allows you to configure:

  • An OPC UA Server, that can be accessed by a Classic OPC Client.

  • An OPC Classic DA 2.05a Server, that can be accessed by a UA Client.

Note: You must restart the Service whenever you make any changes to the gateway.
