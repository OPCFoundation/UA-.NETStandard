OPC Foundation UA Reference Server

Introduction:

This OPC Server is designed to be default OPC UA Server when opening the OPC UA Compliance Test Tool (www.opcfoundation.org/uactt/) and uses an address-space that matches the design of the CTT.

This OPC Server is built with the Quickstart application template (OPC Server) and uses the OPC Foundation .NET UA SDK. This OPC Server therefore supports the opc.tcp and http transports.

Download the CTT from: http://www.opcfoundation.org/uactt/
